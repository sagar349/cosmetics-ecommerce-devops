# Terraform Setup for Cosmetics E-Commerce Project

This folder contains Terraform files to provision:

- AWS EC2 instance for web server
- AWS EC2 instance for Jenkins CI/CD
- AWS EKS cluster for containerized services

Variables are defined in `variables.tf`.  
Outputs for resources are defined in `outputs.tf`.

> Note: AMIs, subnets, and IAM roles are placeholders for demonstration purposes.
